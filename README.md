📌 Project Overview
This repository contains a dynamic, high-performance Professional Portfolio Dashboard built using ASP.NET Core 10.0 MVC. The project is designed to showcase my academic journey, technical stack, and personal projects in a clean, modern, and interactive interface.

The application is fully responsive, featuring smooth animations (Animate.css) and a modular architecture that separates data logic from the visual presentation.

🛠️ Tech Stack
Backend: ASP.NET Core 10.0 (MVC Pattern)

Frontend: C# Razor Pages, HTML5, CSS3

Styling: Bootstrap 5 & Animate.css

Icons: Bootstrap Icons

Deployment: MonsterASP.NET (IIS Environment)

🌟 Key Features
Dynamic CV Rendering: Utilizes a custom CvViewModel to inject data into the dashboard.

Interactive UI: Hover-responsive project cards and "slide-in" entry animations for better UX.

Document Access System: A built-in Modal system allowing recruiters to request official transcripts and certificates.

Technical Proficiency Bars: Visual representation of skills categorized by competency levels.

📁 Repository Structure
/Controllers: Handles the routing and logic for downloading the CV and requesting documents.

/Models: Contains the CvViewModel which defines the structure of the portfolio data.

/Views: Contains the Index.cshtml dashboard and shared layouts.

/wwwroot: Houses all static assets including custom CSS and images.

🚀 Deployment Info
The site is compiled and published using the Release configuration and is currently hosted on a Windows IIS environment.
